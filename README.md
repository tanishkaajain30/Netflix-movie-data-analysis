# Netflix Movie Data Analysis

## Project Overview
This project performs Exploratory Data Analysis (EDA) on the Netflix Movies and TV Shows dataset to uncover trends in content production, release patterns, genres, ratings, and audience preferences.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Objectives
- Analyze Netflix content distribution.
- Identify yearly trends in movie and TV show releases.
- Explore content ratings and genres.
- Understand audience engagement patterns.

## Key Insights
- Netflix content releases increased significantly over recent years.
- Movies constitute a larger share of Netflix content than TV Shows.
- Certain genres and ratings dominate the platform's catalog.
- Content production varies across countries and years.

## Analysis Performed
- Data Cleaning and Preprocessing
- Missing Value Handling
- Content Type Analysis
- Year-wise Trend Analysis
- Genre Distribution Analysis
- Rating Analysis
- Data Visualization
